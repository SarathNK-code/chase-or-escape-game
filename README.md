# Chase & Escape Game

Exported from Caffeine project: Chase & Escape Game

## Quick Start

```bash
# Navigate to frontend directory
cd src/frontend

# Install dependencies
pnpm install

# Start development server
pnpm vite
```

Open http://localhost:5173 to play the game!

## How to Play

- **Escaper**: Collect all 15 coins or survive 90 seconds to win
- **Hunter**: Catch the escaper before time runs out
- **Controls**: 
  - Arrow Keys / WASD - Move your character
  - Space Bar (Escaper only) - Drop decoys to confuse the hunter

## Game Features

- 90-second time limit
- 15 collectible coins
- Smart AI opponent
- Interactive tutorial tour
- User-friendly interface for all ages
